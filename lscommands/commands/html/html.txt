
&lt; !–- -– &gt; For commenting out sections of HTML or placing comments within HTML. Anything between &lt;!-- and --&gt; wont be rendered as text when the page loads in a web browser, but will still be visible if looking at the plain 'ol HTML file in a text editor.

&lt; !DOCTYPE html &gt; Basically tells the browser what kind of document we're loading, in this case good 'ol HTML. This guy likes to be the first element in the document, even before the opening &lt;html&gt; tag. This page, for example, starts things off with &lt;!DOCTYPE html&gt;

&lt; a &gt; a for anchor, also known as a hyperlink, or simply a link. To make an actual link using the &lt;a&gt; tag, use the href attribute: &lt;a href="http://eastmanreference.com"&gt;Link&lt;/a&gt;

&lt; abbr &gt; Abbr is an abbreviation for abbreviation. Use it with the title attribute to define abbreviations. &lt;abbr title="Cascading Style Sheets"&gt;CSS&lt;/abbr&gt;. Safari 12.0 hovering the mouse over CSS brings up a little tool tip generated by the abbreviation tag.

&lt; address &gt; Contact information for the author. Alternatively, this could be general contact information for the website.

&lt; area &gt; Establishes areas in an image map. &lt;area&gt; is always used inside of &lt;map&gt;.

&lt; article &gt; The &lt;article&gt; tag is new in HTML5. This tag can be used to contain blog entries, forum posts, etc.

&lt; aside &gt; New in HTML5. The &lt;aside&gt; element is for making sidebars that are placed next to the main content.

&lt; audio &gt; New in HTML5. The &lt;audio&gt; tag can be used to place an audio stream within an HTML document.

&lt; b &gt; For bolding or otherwise styling text without conveying any additional meaning such as with &lt;em&gt; and &lt;strong&gt; and etc.

&lt; base &gt; Use to set a base URL.

&lt; bdi &gt; Short for bidirectional isolation. Useful for working with languages that use right to left text.

&lt; bdo &gt; Short for Bidirectional Override. Use this to override the current direction of text. Useful for languages that use right to left text.

&lt; blockquote &gt; For quoting from an external source. Typically rendered as indented text.

&lt; body &gt; Establishes the body of an HTML document.

&lt; br &gt; Single line break. Think of br as short for break.

&lt; button &gt; Specifies button that can be clicked. Commonly used with forms.

&lt; canvas &gt; Used for rendering graphics on the fly, typically in conjunction with a scripting language.

&lt; caption &gt; Placed just after &lt;table&gt;, used for table caption.

&lt; cite &gt; For citing.

&lt; code &gt; For code examples. A little bit like &lt;samp&gt; and &lt;kbd&gt;, but specifically for code.

&lt; col &gt; Specify properties of a &lt;col&gt; within a &lt;colgroup&gt;

&lt; colgroup &gt; Essentially a group of columns.

&lt; data &gt; Used for script friendly data.

&lt; datalist &gt; New in HTML5. Specify predefined data for an &lt;input&gt;.

&lt; dd &gt; For description lists. Use with &lt;dt&gt; and &lt;dl&gt;.

&lt; del &gt; Mark text as deleted without actually deleting it. This will typically be rendered as a strikethrough.

&lt; details &gt; New in HTML5. Can be used for additional details which can be hidden or revealed.

&lt; dfn &gt; Use when first defining a term.

&lt; dialog &gt; Dialog box.

&lt; div &gt; A divider or a kind of generic container.

&lt; dl &gt; Description list. Use with &lt;dd&gt; and &lt;dt&gt;.

&lt; dt &gt; Description list. Use with and &lt;dl&gt;.

&lt; em &gt; Emphasis, which I believe often defaults to italics. For something stronger, use &lt;strong&gt;.

&lt; embed &gt; New in HTML5. A container for external content.

&lt; fieldset &gt; Group related items in a form.

&lt; figure &gt; Define self contained content such as photos or diagrams.

&lt; footer &gt; The footer of a document, below the main content.

&lt; form &gt; Forms for user input.

&lt; h1 &gt; Level 1 heading, the headline or title of a page.

&lt; h2 &gt; Level 2 heading, the subtitles of a page.

&lt; h3 &gt; Level 3 heading.

&lt; h4 &gt; Level 4 heading.

&lt; h5 &gt; Level 5 heading.

&lt; h6 &gt; Level 6 heading.

&lt; head &gt; The head section. Used mostly for metadata.

&lt; header &gt; Not to be confused with &lt;head&gt;, &lt;header&gt; typically contains introductory content and layout that goes above the body.

&lt; hgroup &gt; Used for grouping &lt;h1&gt;, &lt;h2&gt;, &lt;h3&gt;, etc., tags together. Useful for multilevel headings.

&lt; hr &gt; This can be used for separating text within a paragraph. Typically renders as a horizontal line running across the page.

&lt; html &gt; The root level tag of an HTML document. All other HTML tags go inside the HTML tag.

&lt; i &gt; The &lt;i&gt; element has changed a little in HTML5. It can still make text italicized, and I think that is often the default behavior, however, this depends on the stylesheet.

&lt; iframe &gt; Makes it possible to embed another page within a page. In HTML5 this is known as nested browsing.

&lt; img &gt; For displaying images on a webpage. Example: &lt;img src="image.png" &gt;

&lt; input &gt; Input is used with various attributes to create form input elements.

&lt; ins &gt; Useful for denoting text that has been added to an HTML file since the original version. Updates to a blog post for example. This is typically rendered as underlined text, although this can really be rendered just about any way you want with a stylesheet.

&lt; kbd &gt; Used for keyboard input. This is different from form element text input.

&lt; keygen &gt; Used for authentication with a security certificate.

&lt; label &gt; For assigning labels to interface elements such as forms.

&lt; legend &gt; Useful for making forms easier to understand.

&lt; li &gt; This represents a list item with an &lt;ol&gt; (ordered list) or &lt;ul&gt; (unordered list).

&lt; link &gt; Used for linking to external resources. Typically seen in &lt;head&gt; referring to an external style sheet. Different from the &lt;a&gt; (anchor) tag.

&lt; main &gt; New to HTML5. These tags enclose the main content of an HTML document.

&lt; map &gt; Used when defining an image map.

&lt; mark &gt; This is essentially used for highlighting. Of course, whether or not highlighting is actually involved depends on the involved stylesheet.

&lt; menu &gt; Used for creating various kinds of menus. This is new to HTML 5.1.

&lt; menuitem &gt; Specify actions that can be taken with a menu.

&lt; meta &gt; Used for specifying various meta information about the document. Meta tags are used within the document &lt;head&gt;.

&lt; meter &gt; Used to represent a number when the minimum and maximum is known, for example 10%.

&lt; nav &gt; Used for specifying a navigational region within a document.

&lt; noscript &gt; For anyone using scripts within their pages, &lt;noscript&gt; is useful for specifying content that should be rendered when scripting isn’t supported.

&lt; object &gt; For embedding objects within a document.

&lt; ol &gt; Ordered list (1. 2. 3.). Use with list item &lt;li&gt;.

&lt; optgroup &gt; For grouping objects.

&lt; option &gt; Used within the &lt;select&gt; tag to specify an option.

&lt; output &gt; Displays the output of an action, such as from a script or form.

&lt; p &gt; Paragraph. Enclose paragraphs in an article within &lt;p&gt; tags.

&lt; param &gt; Use when parameters need to be passed to an object embedded within the document.

&lt; pre &gt; For preformatted text. This can be useful for demonstrating code, especially if there are multiple lines.

&lt; progress &gt; Display the progress of an action, such as with a script.

&lt; q &gt; Quote from some source.

&lt; rb &gt; For Ruby annotations.

&lt; rp &gt; Specify text to be displayed when Ruby annotations are not supported by the browser.

&lt; rt &gt; Text component of a Ruby annotation.

&lt; rtc &gt; Ruby Text Container.

&lt; ruby &gt; Also used for Ruby annotation.

&lt; s &gt; For marking up text that is not longer relevant, applicable, or accurate.

&lt; samp &gt; Sample output one might get from a particular computer program.

&lt; script &gt; Scripts inside of an HTML document are enclosed in the script tags.

&lt; section &gt; A nonspecific means of breaking up content within a webpage into sections.

&lt; select &gt; Used with &lt;option&gt; for selecting a particular option.

&lt; small &gt; Typically used for small print. Useful for comments within the main content.

&lt; source &gt; Used with &lt;audio&gt; and &lt;video&gt; for specifying media source.

&lt; span &gt; Useful for applying styles to or around text, especially inline.

&lt; strong &gt; The enclosed text is of strong importance.

&lt; style &gt; For declaring style sheets within a document.

&lt; sub &gt; For displaying subscript. Useful for math, especially in conjunction with &lt;sup&gt; (superscript).

&lt; summary &gt; Used in conjunction with the &lt;details&gt; tag for specifying a summary.

&lt; sup &gt; For displaying superscript. Useful for math, especially in conjunction with &lt;sub&gt; (subscript).

&lt; table &gt; For making tables in an HTML page.

&lt; tbody &gt; Groups table rows.

&lt; td &gt; Table data; this represents a data cell within a table.

&lt; template &gt; Declares HTML snippets that can be used later.

&lt; textarea &gt; For multi-line text entry.

&lt; tfoot &gt; Table footer.

&lt; th &gt; Table header cell.

&lt; thead &gt; Table header row.

&lt; time &gt; Specifies the date/time within a document.

&lt; title &gt; The title of an HTML document.

&lt; tr &gt; A row within a table.

&lt; track &gt; Used to specify text tracks with &lt;audio&gt; and &lt;video&gt;.

&lt; u &gt; Useful for annotating text for various reasons. No longer specifically for underlining, although you can of course still do that with CSS.

&lt; ul &gt; Unordered list. Use with &lt;li&gt; (list item) to make unordered lists.

&lt; var &gt; Defines a variable within the content of an HTML page.

&lt; video &gt; For embedding video within an HTML page.

&lt; wbr &gt; The optional line wrap.
